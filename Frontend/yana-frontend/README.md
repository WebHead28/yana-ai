# YANA Frontend

You Are Not Alone — React + Vite + TailwindCSS frontend.

## Setup

```bash
npm install
npm run dev
```

## Environment

Create a `.env` file if your backend runs on a different port:

```
VITE_API_URL=http://localhost:8000
```

If left empty, requests will be proxied via Vite's dev proxy to `http://localhost:8000`.

## Build

```bash
npm run build
```

## Stack

- React 18
- Vite 5
- TailwindCSS 3
- Axios
- React Router 6
